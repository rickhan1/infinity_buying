# Directives (Layer 1: What to do)

이 디렉토리에는 SOP(Standard Operating Procedure)가 Markdown 형식으로 저장됩니다.

## 역할
- 목표, 입력, 사용할 도구/스크립트, 출력, 엣지 케이스를 정의
- 자연어로 작성된 지시사항 (중급 직원에게 건네줄 수 있는 수준)
- 학습 내용이 발생하면 지속적으로 업데이트되는 살아있는 문서

## 규칙
- 새로운 directive를 생성하거나 덮어쓰기 전에 항상 사용자에게 확인
- API 제약, 더 나은 접근법, 공통 에러, 타이밍 기대치 등을 발견하면 업데이트
