# 무한매수법 (Infinite Buying Method) SOP

## 전략 개요
미국 ETF(TQQQ, SOLX)에 대해 40분할 LOC 매수를 통해 평단가를 관리하며, +10% 익절 / -10% 손절 시 사이클을 리셋하는 기계적 매매 전략.

## 설정값
- **종목**: TQQQ, SOLX (각각 독립 운용)
- **초기 자금**: 각 $100,000
- **분할 수**: 40
- **1회 매수금**: 초기자금 / 분할수 = $2,500
- **알파(α)**: 3% (config.py에서 변경 가능)
- **익절선**: +10%
- **손절선**: -10%

## 매수 로직 (LOC 방식)

### T=1 (첫 매수)
- 종가 기준 전액($2,500) 매수

### T≥2 (이후 매수)
1. **종가 ≤ 평단가**: 전액($2,500) 매수 체결
2. **평단가 < 종가 ≤ 평단가 × (1 + α)**: 절반($1,250)만 매수 체결
3. **종가 > 평단가 × (1 + α)**: 매수 미체결 → 금액은 현금 잔고에 유지

### 미체결 금액 처리
-  현금 잔고에 누적 유지
- 40분할 이후에도 사용 가능

## 매도/사이클 로직
1. **익절**: 현재 평가금 ≥ 총 매수금 × 1.10 → 전량 매도, 사이클 리셋
2. **손절**: 현재 평가금 ≤ 총 매수금 × 0.90 → 전량 매도, 사이클 리셋
3. **사이클 리셋**: 현금 잔고를 초기 자금으로 복원, 사이클 번호 증가

## 계산 공식
- **평단가** = 총 매수금액 / 총 보유수량
- **평가금** = 보유수량 × 현재 종가
- **수익률** = (평가금 - 총 매수금액) / 총 매수금액 × 100

## 엣지 케이스
- 주말/공휴일: 매수 없음 (거래일만 실행)
- 40회 매수 완료 후: 추가 매수 없이 익절/손절 대기
- 현금 잔고 부족: 잔고 범위 내에서만 매수
